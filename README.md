# Supply Chain Correlation Analysis

**Author:** 23f2005282@ds.study.iitm.ac.in

## Project Overview

This project analyzes correlations between supply chain metrics for OptimalFlow Logistics.

## Dataset Variables

- Supplier_Lead_Time: Days from order placement to delivery
- Inventory_Levels: Current stock quantities (units)
- Order_Frequency: Number of orders placed per month
- Delivery_Performance: On-time delivery rate (%)
- Cost_Per_Unit: Unit cost in dollars ($)

## Files

1. **correlation.csv** - 5x5 correlation matrix
2. **heatmap.png** - Excel conditional formatting visualization (Red-White-Green)
3. **README.md** - This file

## Analysis Method

Generated using Excel Data Analysis ToolPak:
- Data → Data Analysis → Correlation
- Applied Red-White-Green conditional formatting
- Matrix shows correlations between all variable pairs

## Key Findings

The correlation matrix reveals relationships between supply chain metrics to optimize supplier selection and inventory management.
